## Instalation

### To install ProteinMPNN:

- Install [conda](https://docs.conda.io/en/latest/)
- Clone this repo
- `cd` to the repo folder
- Run `conda env create --file environment.yml`

### To run ProteinMPNN

- Run `conda activate protein_mpnn`
- Run `protein_mpnn --help`
